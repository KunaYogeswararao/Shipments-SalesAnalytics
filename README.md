Welcome to this Power BI Data Analytics project! In this video, I present a fully interactive Shipment & Sales Performance Dashboard, designed to track product-wise shipment details, sales contribution, profit percentage, and salesperson performance. This dashboard provides actionable insights that help businesses understand:
✔️ Shipment trends over time
✔️ Best-performing products
✔️ Total profit and profit % comparison
✔️ Sales by region (Geo analysis)
✔️ Top-performing salespersons
✔️ Shipment box distribution and metrics
